Current features and problems:
1.Game name: Guess a number between 1 and 100.
2.Game description: The user have to guess a random number between 1 and 100, user have only 8 tips to win the game. If user misses all of the tips, user loses the game. The program asks user about a new game or exit and contains protectors in case of incorrect instructions. Furthermore the game helps user if the number is too low or high.
3.Bugs and problems: If the user win the game, and it restart, then the program does not generate a new number.
